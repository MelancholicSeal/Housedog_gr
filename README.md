# Housedog_gr
Εργασία Κατανεμημένα 2024-25

## Installation && Execution

1. **Clone Repo:**
```bash
git clone https://github.com/MelancholicSeal/Housedog_gr
```
2. **Run Backend**
   In projects directory
```bash
mvn spring-boot:run
```
3. **Run Frontend**
   In FrontEnd2 sub-directory
```bash
npm install
npm run dev
```
